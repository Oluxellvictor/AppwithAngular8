JWT Authentication
  • JSON Web Tokens are an open, industry standard RFC 7519 method for representing claims securely between two parties.
  • JWT.IO allows you to decode, verify and generate JWT.
  • JWTs can be signed using a secret (with the HMAC algorithm) or a public/private key pair using RSA or ECDSA.
  • JWTs Signed tokens can verify the integrity of the claims contained within it.

Asp.Net Identity Framework
  • ASP.NET Core Identity is a membership system that adds login functionality to ASP.NET Core apps.
  • Users can create an account with the login information stored in Identity or they can use an external login provider.
  • External login providers include Facebook, Google, Microsoft Account, and Twitter.
  • Identity can be configured using a SQL Server database to store user names, passwords, and profile data.
